# Project dependencies

This project is a React + Vite app.

## Install dependencies

Run this from project root:

```bash
npm install
```

## Required dependencies (in package.json)

- react
- react-dom
- react-router-dom
- recharts
- lucide-react
- react-hot-toast
- @tailwindcss/vite
- tailwindcss
- postcss
- autoprefixer

## Dev dependencies

- vite
- @vitejs/plugin-react
- eslint
- @eslint/js
- eslint-plugin-react-hooks
- eslint-plugin-react-refresh
- @types/react
- @types/react-dom
- globals

## Add missing packages

If you added an icon from `react-icons`, install it too:

```bash
npm install react-icons
```

## Quick start

1. Clone repository
2. Run `npm install`
3. Run `npm run dev`
4. Open network URL (usually http://localhost:5173)

## Windows PowerShell execution policy (if required)

If `npm` is blocked in PowerShell, run in a normal command-line shell (cmd) or set policy:

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

# Note

Keep `package.json` in sync when adding or removing packages.
